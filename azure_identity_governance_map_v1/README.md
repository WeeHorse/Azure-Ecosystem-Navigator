# Azure Identity & Governance Concept Map

This pack contains:
- `diagram.md` — Mermaid overview with clickable nodes
- `concepts/*.md` — concept pages with short explanations and official Microsoft Learn links

Suggested use:
1. Open `diagram.md` in VS Code, GitHub, or Azure DevOps Wiki
2. Click a node to open the concept card
3. Use the official documentation links at the bottom of each concept card

This map is designed to complement the Azure Security Controls map.
