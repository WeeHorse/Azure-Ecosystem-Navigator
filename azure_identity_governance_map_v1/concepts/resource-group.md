# Resource Group

## What it is
A resource group is a logical container for Azure resources.

## Why it matters
It is often used as an operational scope for permissions, policies, and lifecycle management.

## Security relevance
- Common scope for delegated access
- Helps structure environments
- Useful for governance and organization

## Related concepts
- Subscription
- RBAC
- Azure Policy
