# Key Vault

## What it is
Key Vault stores secrets, keys, and certificates for applications and services.

## Why it matters
It is a common destination for workload authentication and secret retrieval.

## Security relevance
- Centralized secret storage
- Controlled access via identities
- Supports more secure application design

## Related concepts
- Managed Identity
- RBAC
