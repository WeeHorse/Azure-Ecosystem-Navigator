# Group

## What it is
A group is a collection of identities used to manage access and administration more efficiently.

## Why it matters
Assigning access to groups is usually more manageable than assigning access to users one by one.

## Security relevance
- Supports scalable access management
- Reduces manual permission handling
- Helps structure least-privilege administration

## Related concepts
- Microsoft Entra ID
- RBAC
