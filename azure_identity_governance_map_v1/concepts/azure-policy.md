# Azure Policy

## What it is
Azure Policy is a governance service used to assess and enforce rules across Azure resources and scopes.

## Why it matters
It helps standardize environments and reduce drift.

## Security relevance
- Enforces security baselines
- Supports compliance and governance
- Useful across subscriptions and resource groups

## Related concepts
- Subscription
- Resource Group
- RBAC
