# Managed Identity

## What it is
A managed identity is an Azure identity assigned to a workload or service.

## Why it matters
It lets applications authenticate without storing secrets in code or configuration.

## Security relevance
- Reduces credential exposure
- Supports least privilege
- Commonly used with Key Vault

## Related concepts
- Microsoft Entra ID
- Key Vault
