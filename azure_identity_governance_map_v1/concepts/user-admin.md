# User / Admin

## What it is
A user or administrator is a human identity that signs in and performs actions in Azure and related services.

## Why it matters
Users are the starting point for many access-control and governance decisions.

## Security relevance
- Should use least privilege
- High-privilege accounts should be tightly controlled
- Often protected with Conditional Access and PIM

## Related concepts
- Microsoft Entra ID
- RBAC
- PIM
- Conditional Access
