# Microsoft Entra ID

## What it is
Microsoft Entra ID is the identity system commonly used for authentication and identity administration in Azure environments.

## Why it matters
It is the foundation for sign-in, group membership, application identities, and many access-control flows in Azure.

## Security relevance
- Central identity provider
- Basis for RBAC, Conditional Access, and Managed Identity
- Important in both cloud-only and hybrid identity designs

## Related concepts
- User / Admin
- Group
- RBAC
- Conditional Access
- Managed Identity
