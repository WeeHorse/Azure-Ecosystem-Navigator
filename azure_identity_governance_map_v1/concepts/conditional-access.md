# Conditional Access

## What it is
Conditional Access evaluates sign-in conditions and enforces access requirements.

## Why it matters
It is commonly used to reduce risk around user sign-ins.

## Security relevance
- MFA enforcement
- Risk-based access decisions
- Device and location conditions

## Related concepts
- Microsoft Entra ID
- User / Admin
