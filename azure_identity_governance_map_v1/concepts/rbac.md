# RBAC

## What it is
Azure RBAC is the authorization model that controls who can do what on which scope.

## Why it matters
It is one of the core mechanisms for delegating Azure administration.

## Security relevance
- Enforces least privilege
- Applies at management group, subscription, resource group, and resource scope
- Central to secure operations

## Related concepts
- Microsoft Entra ID
- Subscription
- Resource Group
- PIM
