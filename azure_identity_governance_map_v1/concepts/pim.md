# PIM

## What it is
Privileged Identity Management helps control, review, and time-limit privileged access.

## Why it matters
It reduces the risks of permanent standing privileges.

## Security relevance
- Just-in-time elevation
- Approval and review flows
- Better control of privileged roles

## Related concepts
- RBAC
- User / Admin
- Microsoft Entra ID
