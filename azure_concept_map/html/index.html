Concept cards

  • Tenant — Top-level identity boundary in Azure, tied to a Microsoft Entra directory..
  • Management Group — Hierarchy level above subscriptions for consistent policy and access management..
  • Subscription — Administrative and billing boundary that contains Azure resources..
  • Resource Group — Container for related Azure resources, often grouped by lifecycle..
  • Microsoft Entra ID — Cloud identity and access management service for users, groups, apps, and devices..
  • Azure RBAC — Authorization model that controls who can do what at which Azure scope..
  • Azure Policy — Governance service that evaluates and enforces rules across Azure resources..
  • Virtual Network (VNet) — Fundamental private networking construct for Azure workloads..
  • VPN Gateway — Encrypted connectivity between Azure VNets and on-premises or between VNets..
  • ExpressRoute — Private connection from on-premises infrastructure into Microsoft cloud services..
  • Key Vault — Secure store for secrets, keys, and certificates..
  • Azure Monitor — Unified observability service for metrics, logs, traces, and events from cloud and hybrid resources..
  • Defender for Cloud — Cloud security posture and threat protection across Azure and hybrid workloads..
  • Azure Arc — Extends Azure management, governance, and security to on-premises and multicloud resources..
  • On-premises Environment — Local datacenter or corporate infrastructure outside Azure..
