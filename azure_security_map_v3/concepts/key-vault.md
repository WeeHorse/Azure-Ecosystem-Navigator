# Key Vault

## What it is
Stores secrets, keys, and certificates securely.

## Security relevance
- Central secret management
- Prevents credential leakage
- Critical for secure deployments

## Typical relationships
- Accessed via Managed Identity
- Secured with RBAC and networking
- Used by applications
