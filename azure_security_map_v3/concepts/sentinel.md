# Microsoft Sentinel

## What it is
Microsoft Sentinel is a SIEM (Security Information and Event Management – a log collector) and a SOAR (Security Orchestration, Automation, and Response – an automated security handler) platform.

## Security relevance
- Enables threat detection and response
- Correlates signals across sources
- Connects technical data to SOC workflows
- Supports incident handling

## Typical relationships
- Uses data from Azure Monitor
- Correlates logs and alerts
- Supports investigation workflows
