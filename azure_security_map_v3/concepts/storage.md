# Storage Account

## What it is
Provides scalable storage for blobs, files, queues, and tables.

## Security relevance
- Central for data security discussions
- Supports encryption and access control
- Often exposed if misconfigured

## Typical relationships
- Accessed via Private Endpoint
- Monitored via Azure Monitor
- Used by applications

## Official documentation
- https://learn.microsoft.com/en-us/azure/storage/common/storage-introduction

## Tip
Balance between data exposure risk vs secure configuration.
