# App Service

## What it is
App Service is a managed platform for web apps and APIs.

## Security relevance
- PaaS responsibility model
- Connects identity, secrets, and networking
- Common real-world workload

## Typical relationships
- Uses Managed Identity
- Uses Key Vault
- Sends logs to Monitor

## Official documentation
- https://learn.microsoft.com/en-us/azure/app-service/overview

## Tip
Limited security exposure compared to fully managed server.
