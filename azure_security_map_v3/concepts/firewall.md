# Azure Firewall

## What it is
Managed, stateful firewall service for centralized traffic control.

## Security relevance
- Central security control for network flows
- Enables controlled egress/ingress
- Important in zero-trust architecture

## Typical relationships
- Protects traffic inside a VNet
- Works with routing and NSG
- Inspects application-level traffic
