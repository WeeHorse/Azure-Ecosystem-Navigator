# Azure Monitor

## What it is
Azure Monitor collects metrics, logs, and telemetry.

## Security relevance
- Enables detection and troubleshooting
- Critical for operations and security
- Foundation for incident response
- Alerting and troubleshooting
- Can forward data to Sentinel

## Typical relationships
- Collects data from all services
- Feeds into Sentinel
- Supports alerting

## Official documentation
- https://learn.microsoft.com/en-us/azure/azure-monitor/overview

## Tip
Visibility vs control vs response.
